# UX Principles

Visual delight is more important than accuracy.

The first user reaction should be:

"Wow."

Inspired by:

- Apple
- OpenAI
- Midjourney

Rules:

Less text.

More visuals.

Avoid complexity.

Avoid dashboards.

Avoid large reports.

Provide instant feedback.

Use before/after comparisons.

Use beautiful overlays.

Animations should be subtle.

The image should be shareable.

The image itself is the product.
