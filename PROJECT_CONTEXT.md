# Project Context

Project name:

Sixth Sense Vision

Mission:

Turn ordinary photos into extraordinary visual experiences.

Users should feel:

"Wow. This is amazing."

The product should feel magical rather than technical.

The application should help users see things they normally cannot see.

The generated image itself is the product.

Primary focus:

- Beautiful output
- Instant gratification
- Shareable images
- Premium UX

Not primary goals:

- Scientific precision
- Professional diagnosis
- Complex reports

The first version should prioritize visual experience over accuracy.
